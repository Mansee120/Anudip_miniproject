# Anudip_miniproject
🏦 Smart ATM Banking System (Flask + MySQL)
📌 Project Description

The Smart ATM Banking System is a web-based application developed using Python Flask and MySQL that simulates real-world banking operations such as account login, balance management, deposits, withdrawals, fund transfers, transaction history tracking, and analytics visualization.

This system provides a secure and user-friendly interface where users can perform banking activities similar to an ATM or digital banking platform. It also includes enhanced features like OTP-based transaction verification and data visualization for financial insights, making it more advanced than a basic banking system.

🎯 Objective

The main objective of this project is to:

Provide a digital banking experience
Implement secure transaction handling
Demonstrate full-stack development skills
Integrate database management with real-time operations
Offer data analytics for user transactions
⚙️ Technologies Used
Frontend: HTML, CSS (embedded in Flask templates)
Backend: Python (Flask Framework)
Database: MySQL
Visualization: Chart.js (or Matplotlib earlier)
Other: Session handling, OTP logic
🔑 Key Features
🔐 1. Secure Login System
Users log in using a PIN
Session-based authentication ensures security
💰 2. Balance Management
Displays real-time account balance
Updates instantly after transactions
📥 3. Deposit Function
Users can add money to their account
Transactions are recorded in the database
📤 4. Withdrawal Function
Allows users to withdraw money
Includes insufficient balance validation
🔄 5. Fund Transfer (with OTP)
Transfer money to another user via PIN
OTP verification ensures secure transactions
📜 6. Transaction History
Displays all past transactions
Includes type, amount, and date
📊 7. Analytics Dashboard
Visual representation of spending
Pie chart showing transaction distribution
🚪 8. Logout System
Secure session termination
